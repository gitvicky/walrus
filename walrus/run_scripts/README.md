## Run script examples

This are examples of how one can use the Walrus repository. This is a mix of distributed and local jobs.

- eval_onegpu_example_walrus.sh - Evaluates pretrained weights on a dataset using slurm to launch a 1 gpu job
- finetuning_example_distributed_walrus.sh - Finetunes weights for a specific dataset.
- pretrain_example_distributed_walrus.sh - Example script for launching full pretraining. 
- run_local... - Examples of running non-distributed jobs for both Walrus and comparison models.
